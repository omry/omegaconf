# Type Adapter System

This design note has been split into a reviewer-facing RFC and supporting
documents:

- [Type Adapter System RFC](rfc.md) - start here for architecture,
  tradeoffs, public API shape, and open questions.
- [Type Adapter System Implementor Spec](spec.md) - detailed
  mechanics for implementors and maintainers.
- [Type Adapter Author Guide](author-guide.md) - practical guide
  for writing adapter packages.
- [Type Adapter Security And Governance](security-governance.md) -
  ownership tiers, discovery policy, supply-chain boundary, and compatibility
  policy.

The original long-form design content has been moved into the implementor spec
and appendices so the RFC can remain the primary review path.
